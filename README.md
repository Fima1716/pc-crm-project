# Стек проекта CRM для компьютерных мастерских

## Frontend
- **React 18+** — функциональные компоненты, hooks
- **TypeScript** — типизация props, state, функций и API
- **MUI** — готовые таблицы, кнопки и формы
- **React Hook Form + Yup** — валидация форм
- **Redux Toolkit + React-Redux** — глобальный стейт (список заявок, фильтры, статусы)
- **Axios или TanStack Query** — запросы к бэкенду

## Backend
- **Go + Gin или Fiber** — REST API
- **PostgreSQL** 
- **GORM или pgx** — работа с базой данных

## Dev / Локальная разработка
- **GitHub** — контроль версий и репозиторий
- **Postman / Insomnia** — тестирование API

# Стек проекта CRM для компьютерных мастерских

## 🧩 Frontend
- **React 18+** — UI-компоненты и работа с состоянием
- **TypeScript** — типобезопасность и автодополнение
- **React Context** — глобальное состояние (авторизация, текущий пользователь)
- **React Query (@tanstack/react-query)** — кеширование данных с сервера, автоматическое обновление
- **Axios** — HTTP-запросы к API

## 🖥️ Backend
- **Go (Golang)** — быстрый и надёжный сервер, REST API
- **PostgreSQL** — база данных для хранения всех данных (заявки, пользователи и т.д.)

## 🧰 Dev / Инструменты
- **Vite** — сборка и запуск фронтенда
- **GitHub / GitLab** — контроль версий и командная работа
- **Postman / Insomnia** — тестирование API
- **Docker (по желанию)** — контейнеризация сервисов

# Стек проекта CRM для компьютерных мастерских

## 🧩 Frontend
- **React 18+** — UI-компоненты и работа с состоянием
- **TypeScript** — типобезопасность 
- **React Context** — глобальное состояние 
- **React Query (@tanstack/react-query)**
- **Axios** — HTTP-запросы к API

## 🖥️ Backend
- **Go (Golang)**
- **PostgreSQL** — база данных для хранения всех данных (заявки, пользователи и т.д.)
